# Number_app
# Overwiev
 This web application allows users to check various properties of a given number. Users can input a number, and the application will determine if it's even, odd, prime, composite and provide a list of its divisors.
# Getting Started
To run the application, simply open the HTML file in a web browser. No additional setup or installation is required.
# How to Use
Enter a number in the designated input field.
Click the "Check" button to see if the number is even, prime, and to display its divisors.
You will receive clear alerts if the input is invalid.
